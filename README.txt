W : Move forward
S : Move backward
A : Move to left
D : Move to right

Left Arrow  : Size up the map view
Right Arrow : Size up the 3D view
Up Arrow    : Increase the density
Down Arrow  : Decrease the density

NUMPAD 8 : Increase the map's height
NUMPAD 6 : Increase the map's width
NUMPAD 4 : Decrease the map's height
NUMPAD 2 : Decrease the map's width

T : Switch mode [Cube | Face]. Default is CUBE.
Face mode isn't finished and it'll remain like that.


     M : Show the matrix.
